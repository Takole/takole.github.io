C# ja sen olioiden ohjelmointia

C# tuli vastaan heti ensimmäisen vuoden syksyllä ja siinä opin perusteita millä logiikalla ohjelmoidaan. Sitä seuraavana keväänä opin siihen liittyvää olio-ohjelmointia. Tämän logiikan ymmärtämisestä on ollut apua muissakin ohjelmointi kielissä.



Kuvassa on Visual Studiossa kirjoitettua koodia C# kielellä. Kyseinen koodin pätkä kysyy käyttäjältä minkä kokoisen taulukon tämä haluaa että tulostetaan konsoliin. Tämän jälkeen taulukko tulostaa käyttäjän antaman kokoisen taulukon. Taulukolla tässä tarkoitetaan numeroita. Seuraavaksi olio-ohjelmointiin liittyviä kuvia:









Kyseessä olio-ohjelmointi kurssin yksi tehtävä ensin on kaksi kuvaa luokista (class), missä määritellään kulkuneuvot ja mitä ominaisuuksia niissä on. Sitten pääohjemassa haetaan classistä ja alustetaan niille tietoja esim. autossa 4 rengasta pyörässä 2. Kuvassa näkyy kuinka jokainen rengas laitetaan foreach-loopilla eli kun tulee vastaan määritetty luku niin ohjelma ei lisää enää renkaita. Viimeinen kuva on mitä tulee näkyviin kun ajaa ohjelman. C# kielessä olen kehittynyt sen perusteista että ymmärrän miten classeja voi käyttää muuallakin hyödyksi.

Mobiiliohjelmointia

Mobiiliohjelmointia olen tehnyt sekä React Nativella että Kotlinilla, sekä siihen liittyen XML:llää. Molempia käynyt yhden kurssin ja React Nativella olen tehnyt yhden projektin. Omasta mielestäni pidin enemmän React Nativesta, se tuntui jotenkin sopivammalta itselle.

Kuvat ovat Android Studion kautta pyörivästä emulaattorista otettuja. Siinä olen hommannut WeatherAPI:n OpenWeatherMap:ilta. Ja tämän jälkeen koodannut React Nativella toiminnallisuuden eli oikealta ylhäältä voi lisätä kaupungin. Ja sitten sen voi päivittää että näkee sään tai poistaa.

Tässä näin on React Nativella tekemäni projektin tulosta. Eli se on uutissovellus jossa on eri kategorioita, niistä voi selata itselle mieluisaa. Jos löytää sopivan uutisen niin sitten voi painaa sitä ja se ohjaa sille sivulle mistä uutinen sovellukseen tulee.
